Associate Bonus Automation

Overview: I have created SSIS pipeline which contains almost around 38 SSIS packages which runs togather to calculate the bonus for each associates. It was calculating bonus for almost around 1500 employees and this process runs every after 40 mins to calculate the bonus live through SQL Agent Job and whatever calcualted bonus comes in the table user can see those bonus in UI.(Although UI part is not developed by me.) It involved more than 30 tables out of which few are filled automated through SSIS package and it involved large amount of store procedures to calculate the bonus which involve business logic mentioned by client.

Technologies Used

  • SSIS
  • SQL
  • SQL Agent

Features

  • Calculate Associate Bonus Live and Show it to UI
  • Fully Automated Calculation. (No Human Involvment)
  • Correction in the bonus and re run the past bonus data to recalculate if there is any issue in the bonus amount.
